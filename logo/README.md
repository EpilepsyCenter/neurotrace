# TRACER — Vector exports

Open any `.svg` in Affinity Designer (File → Open, or drag in).
Shapes come in fully editable; strokes, colors, curves all preserved.

## Files

- `TRACER-mark.svg` — the Scope icon, ink on transparent (512×512)
- `TRACER-mark-reversed.svg` — Scope on ink background
- `TRACER-lockup-horizontal.svg` — icon + wordmark, side by side
- `TRACER-lockup-stacked.svg` — icon above wordmark, centered
- `TRACER-wordmark.svg` — wordmark only

## Colors (hex)

- Ink `#0F1722`
- Slate `#2B3A4A`
- Blue `#4A6B8A`
- Blue Soft `#6B8FA8`
- Mist `#E8EDF2`
- Paper `#FAFAF7`

## Type

Wordmark is **IBM Plex Sans**, weights **300** (Neuro) + **500** (Trace),
letter-spacing −1.5%.

Install Plex Sans (free, Google Fonts) before opening if you want live text.
Otherwise, in Affinity: select the text, **Text → Convert to Curves** — then
the shapes are fully editable regardless of font availability.

## Recommended workflow

1. Install IBM Plex Sans (https://fonts.google.com/specimen/IBM+Plex+Sans).
2. Open the SVG you want in Affinity Designer.
3. Either leave text live (for edits) or Convert to Curves (for distribution).
4. For the mark, every stroke is editable — adjust weights, colors, or
   simplify the grid/threshold freely.
